🚀 End-to-End Generative AI Application

📌 Overview

This project is an end-to-end Generative AI application built using LangChain that summarizes content from YouTube videos and web pages. It leverages the Grok API with open-source LLMs to extract, process, and generate concise summaries from long-form content.

The application is designed to improve information consumption by transforming lengthy content into quick, digestible insights.

⸻

✨ Features

🎥 YouTube Video Summarization
	•	Accepts a YouTube URL as input
	•	Extracts video transcripts automatically
	•	Generates concise summaries highlighting key points

🌐 Website Content Summarization
	•	Takes any webpage URL
	•	Scrapes and processes textual content
	•	Produces structured summaries

🧠 LLM Integration
	•	Integrated with Grok API for model inference
	•	Uses Gamma (Google) model for high-quality summarization
	•	Flexible architecture to experiment with multiple LLMs

⸻

🏗️ Tech Stack
	•	Python
	•	LangChain
	•	Requests / Web Scraping Libraries
	•	Grok API (LLM Integration)

⸻

⚙️ Getting Started

📋 Prerequisites
	•	Python 3.x
	•	Pip package manager

Install required dependencies:

pip install -r requirements.txt


⸻

🔧 Installation

Clone the repository:

git clone https://github.com/DhanushRathod15/Text-Summarization.git

Navigate to the project directory:

cd gen-ai-summary-app

Set up environment variables:
	•	Create a .env file
	•	Add your Grok API Key

Example:

GROK_API_KEY=your_api_key_here


⸻

▶️ Usage

Run the application:

python app.py

Steps:
	1.	Enter a YouTube URL or Website URL
	2.	Click on Summarize
	3.	View the generated concise summary
